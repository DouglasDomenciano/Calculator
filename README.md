# Calculator
This is a small calculator app written with react-native, used for studying e testing knowledge and skills to prepare for on the future career on full-stack react developer!

## How to Run

> after you [configure](https://facebook.github.io/react-native/docs/getting-started.html) your environment to run a react-native project, so follow this steps:

OBS: Follow the REACT NATIVE CLI QUICKSTART tab for run this project.

### Start your AVD (Android Virtual Device)

```console
foo@bar:~$ emulator -no-snapshot -avd YOUR_AVD_NAME
```

### Enter in folder project

```console
foo@bar:~$ cd Calculator
```

### And run...

```console
foo@bar:~$ cd react-native run-android
```
